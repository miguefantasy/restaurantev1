# Miguel Barroso Dueñas
Aplicacion restaurantev1
Es una aplicacion con 4 pantallas
1.-ConfigScreen
  Aqui elegiremos el numero de mesas del restaurante
2.-MenuScreen
  Aqui crearemos la carta del menu, podremos añadir o eliminar platos
3.-MesasScreen
  Donde veremos las mesas que hay
  Intentaremos hacerlo interactivo para que nos lleve a la cuenta de cada una de las mesas
4.-CuentaScreen
  Aqui estara la cuenta de la mesa seleccionada y podremos ver los productos elegidos y sus precios
